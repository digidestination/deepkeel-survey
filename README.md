# DeepKeel Survey

Professional sailboat self-survey and negotiation system.

## Live URL
https://deepkeel.ivansorkin.com

## Local run (Docker)
```bash
docker compose up -d --build
```

App listens on:
- http://localhost:9890

## Included MVP modules
- Login (day one auth)
- Vessel information
- 120-point checklist scaffold
- Risk score (BCI)
- Negotiation calculator
- Survey report page
